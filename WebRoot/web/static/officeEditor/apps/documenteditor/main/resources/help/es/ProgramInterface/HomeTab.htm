Pestaña de Inicio

La pestaña de Inicio se abre por defecto cuando abre un documento. Le permite cambiar el tipo de letra y los párrafos. También hay más opciones disponibles como Combinación de Correspondencia, combinación de colores y ajustes de vista.

Al usar esta pestaña podrás:

  • establecer el tipo,tamaño y color de letra,
  • aplicar estilos decorativos a la letra,
  • seleccionar color de fondo para un párrafo,
  • crear listas con puntos y numeradas,
  • cambiar las sangrías de un párrafo,
  • establecer el espaciado de línea de un párrafo,
  • alinear su texto en un párrafo,
  • mostrar/ocultar caracteres no imprimibles,
  • copiar/borrar el formato de texto,
  • cambiar la combinación de colores,
  • usar la Combinación de Correspondencia,
  • gestionar estilos,
  • ajustar los Ajustes de Vista y acceder al editor de Ajustes Avanzados.
