Añadir bordes

Para añadir bordes a un párrafo, una página o a un documento,

  1. Ponga el cursor en el párrafo que usted necesita, o elija varios párrafos usando el ratón, o todo el texto pulsando combinación de teclas Ctrl+A,
  2. haga clic derecho y elija la opción Ajustes de párrafo avanzados en el menú, o use el enlace Mostrar ajustes avanzados en la barra derecha lateral,
  3. pase a la sección Bordes y relleno en la ventana Párrafo - ajustes avanzados,
  4. establezca el valor necesario para Tamaño de borde y elija Color de borde,
  5. pulse un diagrama disponible o use botones para seleccionar bordes y aplique el estilo seleccionado,
  6. pulse el botón OK.

Después de añadir los bordes puede fijar los márgenes, es decir, la distancia entre los bordes derecho, izquierdo, superior, inferior y el texto de párrafo dentro ellos.
