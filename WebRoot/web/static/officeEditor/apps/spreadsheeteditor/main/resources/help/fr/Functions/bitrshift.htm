Fonction BITDECALD

La fonction BITDECALD est l'une des fonctions d'ingénierie. Elle est utilisée pour renvoyer un nombre décalé à droite du nombre de bits spécifié.

La syntaxe de la fonction BITDECALD est :

BITDECALD(number, shift_amount)

où

number est un nombre entier supérieur ou égal à 0,

shift_amount est un nombre de bits par lequel vous voulez décaler number, un nombre entier.

Les valeurs numériques peuvent être saisies à la main ou incluses dans les cellules auxquelles il est fait référence.

Décaler un nombre à droite équivaut à retirer des chiffres à droite de la représentation binaire du nombre.

Pour appliquer la fonction BITDECALD,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Ingénierie depuis la liste,
  4. cliquez sur la fonction BITDECALD,
  5. insérez les arguments nécessaires en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
