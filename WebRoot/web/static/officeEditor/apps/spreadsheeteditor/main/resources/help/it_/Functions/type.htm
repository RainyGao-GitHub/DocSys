Funzione TYPE

La funzione TYPE è una delle funzioni della categoria Informazione. Si usa per determinare il tipo di un valore ricevuto o visualizzato.

La sintassi della funzione TYPE è:

TYPE(value)

dove value è un valore da verificare inserito a mano o incluso nella cella alla quale fate riferimento. Qui sotto trovate i valori possibili e il risultato che la funzione TYPE restituisce:

Valore Risultato
numero 1
testo 2
valore logico 4
valore di errore 16
freccia 64

Per applicare la funzione TYPE,

  1. selezionate la cella dove desiderate visualizzare il risultato,
  2. cliccate sull'icona Inserisci funzione sulla barra degli strumenti superiore,
    o cliccate con il tasto destro del mouse sulla cella scelta e selezionate l'opzione Inserisci funzione dal menu contestuale,
    o cliccate sull'icona prima della barra della formula,
  3. selezionate il gruppo di funzioni Informazione dall'elenco,
  4. cliccate sulla funzione TYPE,
  5. inserite l'argomento richiesto,
  6. premete il tasto Enter.

Il risultato sarà visualizzato nella cella scelta.
