Fonction PAS

La fonction PAS est l'une des fonctions logiques. Elle sert à vérifier si la valeur logique saisie est vraie ( TRUE ) ou fausse ( FALSE ). La fonction renvoie vrai ( TRUE ) si l'argument est faux ( FALSE ) et renvoie faux ( FALSE ) si l'argument est vrai ( TRUE ).

La syntaxe de la fonction PAS est :

PAS(logical)

où logical est une valeur saisie à la main ou incluse dans la cellule à laquelle il est fait référence.

Pour appliquer la fonction PAS,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Logiques depuis la liste,
  4. cliquez sur la fonction PAS,
  5. saisissez l'argument requis,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
