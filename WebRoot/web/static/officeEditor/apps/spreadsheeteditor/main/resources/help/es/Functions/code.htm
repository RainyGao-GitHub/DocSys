Función CODIGO

La función CODIGO es una función de texto y datos. Se usa para devolver el valor ASCII de un carácter especificado o el primer carácter en una celda.

La sintaxis de la función CODIGO es:

CODIGO(cadena)

donde cadena es un dato introducido manualmente o incluido en la celda a la que usted hace referencia.

Para aplicar la función CODIGO,

  1. seleccione la celda donde usted quiere mostrar el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de funciones Texto y datos en la lista,
  4. haga clic en la función CODIGO,
  5. introduzca el argumento correspondiente,

    Nota: la función CODIGO es sensible a mayúscula y minúscula.

  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
