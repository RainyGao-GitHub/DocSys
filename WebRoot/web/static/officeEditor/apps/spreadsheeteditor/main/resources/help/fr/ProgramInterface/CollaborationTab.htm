Onglet Collaboration

L'onglet Collaboration permet d'organiser le travail collaboratif sur le classeur: partage du fichier, sélection d'un mode de co-édition, gestion des commentaires.

En utilisant cet onglet, vous pouvez :

  • spécifier les paramètres de partage,
  • basculer entre les modes de coédition Strict et Rapide,
  • ajouter des commentaires au document,
  • Ouvrir le panneau Chat.
