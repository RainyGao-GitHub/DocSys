Función PRONOSTICO.LINEAL

La función PRONOSTICO.LINEAL es una función estadística. Se usa para calcular, o predecir, un valor futuro usando valores existentes; el valor previsto es un valor y por cada valor x. Los valores conocidos son valores x y y existentes, y el valor nuevo se predice usando una regresión linear.

La sintaxis de la función PRONOSTICO.LINEAL es:

PRONOSTICO.LINEAL(x, ys_conocidas, xs_conocidas)

donde

x es un valor x usado para pronosticar el valor y, un valor numérico introducido manualmente o incluido en la celda a la que usted hace referencia.

ys_conocidas es un conjunto de valores «y» conocidos.

xs_conocidas es un conjunto de valores «x» conocidos.

Para aplicar la función PRONOSTICO.LINEAL,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de funciones Estadísticas en la lista,
  4. haga clic en la función PRONOSTICO.LINEAL,
  5. introduzca los argumentos correspondientes separados por comas,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
