Fonction RACINE.PI

La fonction RACINE.PI est l'une des fonctions mathématiques et trigonométriques. Elle est utilisée pour renvoyer la racine carrée de la constante pi (3.14159265358979) multipliée par le nombre spécifié.

La syntaxe de la fonction RACINE.PI est :

RACINE.PI(x)

où x est un nombre que vous souhaitez multiplier par pi, une valeur numérique saisie à la main ou incluse dans la cellule à laquelle il est fait référence.

Remarque: si x (nombre) est négatif, la fonction renvoie l'erreur #NOMBRE!.

Pour appliquer la fonction RACINE.PI,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône
    sutuée sur la barre de formule,
  3. sélectionnez le groupe de fonctions Maths et trigonométrie depuis la liste,
  4. cliquez sur la fonction RACINE.PI,
  5. saisissez l'argument requis,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
