Limpie texto, formatee en una celda, copie formato de celda

Elimine formato

Usted puede eliminar rápidamente un texto o un formato dentro de una celda seleccionada. Para hacerlo,

  1. seleccione una celda, un rango de celdas usando el ratón o toda la hoja de cálculo pulsando la combinación de las teclas Ctrl+A,

    Nota: también puede seleccionar varias celdas o rangos de celdas no adyacentes pulsando la combinación de las teclas Ctrl mientras selecciona las celdas/rangos con el ratón.

  2. pulse el icono Limpiar en la pestaña de Inicio en la barra de herramientas superior y seleccione una de las opciones disponibles:
    • use la opción Todo si quiere eliminar un texto, un formato, una función etc;
    • use la opción Texto si quiere eliminar un texto del rango de celdas seleccionado;
    • use la opción Formato si usted quiere limpiar un formato del rango de celdas seleccionado. Si hay un texto o funciones, no cambiarán;
    • use la opción Comentarios si usted quiere eliminar comentarios del rango de celdas seleccionado;
    • use la opción Hiperenlaces si usted quiere eliminar hiperenlaces del rango de celdas seleccionado.

    Nota: todas estas opciones están también disponibles en el menú contextual.

Copiar formato de celda

Puede de forma rápida copiar un formato de celda en concreto y aplicarlo a otras celdas.

Para aplicar el formato copiado a una sola celda o varias celdas adyacentes,

  1. seleccione una celda/rango de celdas de la cual usted quiere copiar el formato usnado el ratón o teclado,
  2. pulse el icono Copiar estilo en la pestaña de Inicio en la barra de herramientas superior, (el cursor del ratón estará así ),
  3. seleccione una celda/rango de celdas a las que usted quiere aplicar el formato.

Para aplicar el formato copiado a varias celdas o rangos de celdas no adyacentes,

  1. seleccione una celda/rango de celdas de la cual usted quiere copiar el formato usnado el ratón o teclado,
  2. haga doble clic en el icono Copiar estilo en la pestaña Inicio de la barra de herramientas superior, (el cursor del ratón estará así , y el icono Copiar estilo permanecerá seleccionado: ),
  3. haga clic en celdas o rangos de celdas una por una para aplicar el mismo formato a todas ellas,
  4. Para salir de este modo, haga clic en el icono Copiar Estilo otra vez o presione la tecla Esc en el teclado.
