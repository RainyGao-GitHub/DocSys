Cambie el formato de un número

Aplicar el formato de un número

Usted puede cambiar el formato de un número; por ejemplo, el aspecto de los números introducidos en la hoja de cálculo. Para hacerlo,

  1. seleccione una celda, un rango de celdas usando el ratón o toda la hoja de cálculo pulsando la combinación de las teclas Ctrl+A,

    Nota: también puede seleccionar varias celdas o rangos de celdas no adyacentes pulsando la combinación de las teclas Ctrl mientras selecciona las celdas/rangos con el ratón.

  2. Despliegue la lista Formato de número en la pestaña de Inicio en la barra de herramientas superior y seleccione el formato de número deseado:
    • General - se usa para mostrar datos introducidos como números sin formato de un modo compacto sin ningún signo adicional,
    • Número - se usa para mostrar los números con 0-30 dígitos después de un punto decimal donde un separador de miles se añade entre cada grupo de tres dígitos antes del punto decimal,
    • Exponencial - se usa para mantener cortos a los números convirtiéndoles en una cadena de tipo d.dddE+ddd o d.dddE-ddd donde cada d es un dígito del 0 a 9,
    • Financiero - se usa para mostrar valores monetarios con el símbolo de moneda por defecto y dos sitios de decimales. Para usar otro símbolo de moneda u otro número para los sitios decimales, sigue las instrucciones de más abajo. A diferencia del formato Moneda, el formato Financiero alinea los símbolos de moneda a la parte izquierda de la celda, representa a los valores cero como un guión y los valores negativos de muestran en paréntesis.

      Nota: para poder aplicar de forma rápida el formato Financiero a los datos seleccionados, también puede hacer clic en el icono Estilo financiero en la pestaña de Inicio en la barra de herramientas superior y seleccionar el símbolo de moneda necesario: uno de los símbolos de moneda siguientes: $ Dolar, &euro; Euro, &libra Libra, ₽ Rublo, &yen Yen.

    • Moneda - se usa para mostrar valores monetarios con los símbolos de moneda por defecto y dos lugares de decimales. Para usar otro símbolo de moneda u otro número para los sitios decimales, sigue las instrucciones de más abajo. A diferencia del formato Financiero, en el formato Moneda el símbolo de moneda se coloca antes del primer dígito, los valores negativos se muestran con el signo negativo (-).
    • Fecha - se usa para mostrar fechas,
    • Hora - se usa para mostrar la hora,
    • Porcentaje - se usa para mostrar datos como un porcentaje acompañado por el signo %,

      Nota: para aplicar de forma rápida el formato porcentual a sus datos, también puede usar el icono Estilo porcentaje en la pestaña de Inicio en la barra de herramientas superior.

    • Fracción - se usa para mostrar los números como fracciones comunes en vez de en decimales.
    • Texto - se usa para mostrar los valores numéricos como texto simple y con la mejor precisión posible.
    • Más formatos - se usa para personalizar los formatos de números ya aplicados especificando parámetros adicionales (vea la descripción de más abajo).
  3. cambie el número de posiciones decimales, si es necesario:
    • use el icono Aumentar decimales situado en la pestaña de Inicio en la barra de herramientas superior para mostrar más dígitos después del punto decimal,
    • use el icono Disminuir decimales situado en la pestaña de Inicio en la barra de herramientas superior para mostrar menos dígitos después del punto decimal.

Personalice el formato de los números

Usted puede personalizar el formato de los números que ha añadido de la siguiente manera:

  1. seleccione las celdas donde quiere personalizar el formato de los números para,
  2. desplegar la lista Formato de número en la pestaña de Inicio en la barra de herramientas superior,
  3. seleccione la opción Más formatos,
  4. en la ventana de Formato de número que se abre, ajuste los parámetros disponibles. Las opciones cambian dependiendo del formato de número que se usa en las celdas seleccionadas. Puede usar la lista de Categoría para cambiar el formato del número.

    • para el formato de Número, puede ajustar el número de puntos Decimales, especificar si quiere Usar 1000 separadores o no y elegir uno de los Formatos disponibles para mostrar valores negativos.
    • para los formatos Científico y Porcentaje, puede ajustar el número de puntos Decimales.
    • para los formatos de Financiero y Moneda, puede ajustar el número de puntos Decimales, elegir uno de los Símbolos de moneda disponibles y uno de los Formatos disponibles para mostrar valores negativos.
    • para el formato de Fecha, puede seleccionar uno de los formatos de fecha disponibles: 15/4, 15/4/06, 15/04/06, 15/4/2006, 15/4/06 0:00, 15/4/06 12:00 AM, a, abril 15 2006, 15-abr., 15-abr.-06, abr.-06, abril-06, a-06, 06-abr., 15-abr.-2006, 2006-abr.-15, 06-abr.-15, 15/abr., 15/abr./06, abr./06, abril/06, a/06, 06/abr., 15/abr./2006, 2006/abr./15, 06/abr./15, 15 abr., 15 abr. 06, abr. 06, abril 06, a 06, 06 abr., 15 abr. 2006, 2006 abr. 15, 06 abr. 15, 06/4/15, 06/04/15, 2006/4/15.
    • para el formato de Tiempo, puede seleccionar uno de los formatos de tiempo disponibles: 12:48:58 PM, 12:48, 12:48 PM, 12:48:58, 48:57.6, 36:48:58.
    • para el formato de Fracción, puede seleccionar uno de los formatos disponibles: Hasta un dígito (1/3), Hasta dos dígitos (12/25), Hasta tres dígitos (131/135), Como mitades (1/2), Como cuartos (2/4), Como octavos (4/8), Como dieciseisavos (8/16), Como décimas (5/10) , As centésimas (50/100).
  5. haga clic en el botón OK para confirmar sus cambios.
