Annuler/rétablir vos actions

Pour effectuer les opérations annuler/rétablir, utilisez les icônes correspondantes sur la barre d'outils supérieure :

  • Annuler – utilisez l'icône Annuler pour annuler la dernière action effectuée.
  • Rétablir – utilisez l'icône Rétablir pour rétablir la dernière action annulée.

Remarque: les opérations annuler/rétablir peuvent être effectuées à l'aide des Raccourcis clavier.
