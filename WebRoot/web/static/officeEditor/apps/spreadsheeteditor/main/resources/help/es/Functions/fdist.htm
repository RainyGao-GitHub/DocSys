Función DISTR.F

La función DISTR.F es una función estadística. Se usa para devolver la distribución de probabilidad F (grado de diversidad) para dos conjuntos de datos. Puede usar esta función para determinar si los dos conjuntos de datos tienen distintos grados de diversidad.

La sintaxis de la función DISTR.F es:

DISTR.F(x, grado-libertad1, grado-libertad2)

donde

x es el valor al que la función se debe calcular. Un valor numérico mayor que 0

grado-libertad1 es el grado de numerador de libertad, un valor numérico mayor que 1, pero menor que 10^10.

grado-libertad2 es el grado de denominador de libertad, un valor numérico mayor que 1, pero menor que 10^10.

Los valores pueden introducirse manualmente o incluirse en las celdas a las que usted hace referencia.

Para aplicar la función DISTR.F,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de funciones Estadísticas en la lista,
  4. haga clic en la función DISTR.F,
  5. introduzca los argumentos correspondientes separados por comas,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
