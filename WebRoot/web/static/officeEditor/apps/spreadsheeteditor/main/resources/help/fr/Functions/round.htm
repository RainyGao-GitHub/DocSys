Fonction ARRONDI

La fonction ARRONDI est l'une des fonctions mathématiques et trigonométriques. Elle est utilisée pour arrondir un nombre à un nombre de décimales spécifié.

La syntaxe de la fonction ARRONDI est :

ARRONDI(number, num_digits)

où

number est le nombre à arrondir.

num_digits est le nombre de décimales auquel vous voulez arrondir.

Les valeurs numériques peuvent être saisies à la main ou incluses dans les cellules auxquelles il est fait référence.

Pour appliquer la fonction ARRONDI,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Maths et trigonométrie depuis la liste,
  4. cliquez sur la fonction ARRONDI,
  5. insérez les arguments nécessaires en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
