Fonction EST.PAIR

La fonction EST.PAIR est l'une des fonctions d'information. Elle est utilisée pour vérifier une valeur paire. Si la cellule contient une valeur paire, la fonction renvoie vrai ( TRUE ). Si la valeur est impaire, elle renvoie faux ( FALSE ).

La syntaxe de la fonction EST.PAIR est :

EST.PAIR(nombre)

où nombre est une valeur à tester saisie à la main ou incluse dans la cellule à laquelle il est fait référence.

Remarque: si nombre est une valeur non numérique, EST.PAIR renvoie la valeur d'erreur #VALEUR!.

Pour appliquer la fonction EST.PAIR,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Information,
  4. cliquez sur la fonction EST.PAIR,
  5. saisissez l'argument requis,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
