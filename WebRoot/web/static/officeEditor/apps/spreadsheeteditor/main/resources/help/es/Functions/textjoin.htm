Función UNIRCADENAS

La función UNIRCADENAS es una función de texto y datos. Se usa para combinar el texto de varios rangos y/o cadenas, e incluye un delimitador que especifica entre cada valor de texto que se combinará; si el delimitador es una cadena de texto vacía, la función concatenará los rangos de forma efectiva. Esta función es similar a la función CONCAT, pero la diferencia es que la función CONCAT no puede aceptar un delimitador.

La sintaxis de la función UNIRCADENAS es:

UNIRCADENAS(delimitador, ignorar_vacío, texto1 [, texto2], …)

donde

delimitador es el delimitador que se introduce entre los valores de texto. Se puede especificar como una cadena de texto adjunto en comillas dobles (ej. "," (coma), " " (espacio), "\" (barra invertida) etc.) o como una referencia de una celda o rango de celdas.

ignorar_vacío es un valor lógico que especifica si las celdas vacías deben ignorarse. Cuando el valor se pone en VERDADERO, las celdas vacías se ignoran.

texto1(2) es hasta 252 valores de datos. Cada valor puede ser una cadena de texto o una referencia a un rango de celdas.

Para aplicar la función UNIRCADENAS,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione grupo de funciones Texto y datos en la lista,
  4. haga clic en la función UNIRCADENAS,
  5. introduzca los argumentos correspondientes separados por comas,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
