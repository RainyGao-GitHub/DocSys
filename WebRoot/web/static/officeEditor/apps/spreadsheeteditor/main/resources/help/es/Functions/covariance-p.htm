Función COVARIANZA.P

La función COVARIANZA.P es una función estadística. Se usa para devolver la covarianza de habitantes, la media de la desviación de productos para cada punto de datos en conjuntos de dos datos; usa covarianza para determinar la relación entre dos conjuntos de datos.

La sintaxis de la función COVARIANZA.P es:

COVARIANZA.P(conjunto-1, conjunto-2)

donde conjunto-1(2) es el rango de celdas seleccionado con el mismo número de elementos.

Nota: si conjunto-1(2) contiene texto, valores lógicos o celdas vacías, la función ignorará tales valores, pero procesará las celdas con los valores cero.

Para aplicar la función COVARIANZA.P,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de funciones Estadísticas en la lista,
  4. haga clic en la función COVARIANZA.P,
  5. introduzca los argumentos requeridos separándolos con comas o seleccione el rango de celdas con el ratón,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
