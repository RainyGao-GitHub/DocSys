Funzione TEXT

La funzione TEXT è una delle funzioni della categoria Testo e dati. Si usa per convertire un valore in un testo in un determinato formato.

La sintassi della funzione TEXT è:

TEXT(value, format)

dove

value è un valore da convertire in un testo.

format è il formato di visualizzazione del testo.

I dati possono essere inseriti a mano o inclusi nelle celle alle quali fate riferimento.

Per applicare la funzione TEXT,

  1. selezionate la cella dove desiderate visualizzare il risultato,
  2. cliccate sull'icona Inserisci funzione sulla barra degli strumenti superiore,
    o cliccate con il tasto destro del mouse sulla cella scelta e selezionate l'opzione Inserisci funzione dal menu contestuale,
    o cliccate sull'icona prima della barra della formula,
  3. selezionate il gruppo di funzioni Testo e dati dall'elenco,
  4. cliccate sulla funzione TEXT,
  5. inserite gli argomenti richiesti separati da virgola,
  6. premete il tasto Enter.

Il risultato sarà visualizzato nella cella scelta.
