Fonction PLAFOND.PRECIS

La fonction PLAFOND.PRECIS est l'une des fonctions mathématiques et trigonométriques. Elle est utilisée pour arrondir le nombre à l'excès à l'entier ou au multiple significatif le plus proche. Le nombre est toujours arrondi à l'excès indépendamment de son signe.

La syntaxe de la fonction PLAFOND.PRECIS est :

PLAFOND.PRECIS(x [, précision])

où

x est le nombre à arrondir.

précision est le multiple significatif auquel le nombre doit être arrondi. C'est un paramètre optionnel. En cas d'omission, la valeur par défaut de 1 est utilisée. Si elle est définie sur 0, la fonction renvoie 0.

Les valeurs numériques peuvent être saisies à la main ou incluses dans les cellules auxquelles il est fait référence.

Pour appliquer la fonction PLAFOND.PRECIS,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Maths et trigonométrie depuis la liste,
  4. cliquez sur la fonction PLAFOND.PRECIS,
  5. insérez les arguments nécessaires en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
