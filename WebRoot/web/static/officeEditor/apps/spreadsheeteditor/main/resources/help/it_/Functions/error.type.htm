Funzione ERROR.TYPE

La funzione ERROR.TYPE è una delle funzioni della categoria Informazione. Si usa per restituire la rappresentazione numerica di un errore esistente.

La sintassi della funzione ERROR.TYPE è:

ERROR.TYPE(value)

dove value è un valore di errore inserito a mano o incluso nella cella alla quale fate riferimento. Questo può essere uno dei seguenti valori:

Valore di errore Rappresentazione numerica
#NULL! 1
#DIV/0! 2
#VALUE! 3
#REF! 4
#NAME? 5
#NUM! 6
#N/A 7
#GETTING_DATA 8
Altro #N/A

Per applicare la funzione ERROR.TYPE,

  1. selezionate la cella dove desiderate visualizzare il risultato,
  2. cliccate sull'icona Inserisci funzione sulla barra degli strumenti superiore,
    o cliccate con il tasto destro del mouse sulla cella scelta e selezionate l'opzione Inserisci funzione dal menu contestuale,
    o cliccate sull'icona prima della barra della formula,
  3. selezionate il gruppo di funzioni Informazione dall'elenco,
  4. cliccate sulla funzione ERROR.TYPE,
  5. inserite l'argomento richiesto,
  6. premete il tasto Enter.

Il risultato sarà visualizzato nella cella scelta.
