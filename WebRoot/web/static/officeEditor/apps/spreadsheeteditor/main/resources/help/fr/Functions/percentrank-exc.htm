Fonction RANG.POURCENTAGE.EXCLURE

La fonction RANG.POURCENTAGE.EXCLURE est l'une des fonctions statistiques. Elle est utilisée pour renvoyer le rang d'une valeur dans une série de valeurs en tant qu'un pourcentage(dans l'inervalle ouvert 0..1) de la série.

La syntaxe de la fonction RANG.POURCENTAGE.EXCLURE est :

RANG.POURCENTAGE.EXCLURE(array, x[, significance])

où

array est une plage de cellules sélectionnée contenant des valeurs numériques.

x est la valeur dont le rang vous souhaitez trouver, une valeur numérique saisie à la main ou incluse dans la cellule à laquelle il est fait référence.

significance est le nombre de chiffres significatifs pour le rang à renvoyer. C'est un argument facultatif. S'il est omis, la fonction assume significance d'être égal à 3.

Pour appliquer la fonction RANG.POURCENTAGE.EXCLURE,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Statistique depuis la liste,
  4. cliquez sur la fonction RANG.POURCENTAGE.EXCLURE,
  5. insérez les arguments requis en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
