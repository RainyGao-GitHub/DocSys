Función PRUEBA.Z.N

La función PRUEBA.Z.N es una función estadística. Se usa para devolver el valor absoluto P de una prueba-z. Dada una hipótesis media demográfica, x, la PRUEBA.Z.N devuelve la probabilidad de que el muestreo medio sea mayor que la media de observaciones en el conjunto de datos (matriz) — esto es, el muestreo medio observado.

La sintaxis de la función PRUEBA.Z.N es:

PRUEBA.Z.N(matriz, x [, sigma])

donde

matriz es el rango de valores numéricos contra los que hacer pruebas con x.

x es el valor a poner a prueba.

Sigma es una desviación demográfica estándar. Es un argumento opcional. Si está omitido, se usa la muestra de desviación estándar.

Los datos se pueden introducir manualmente o se pueden incluir en las celdas a las que usted hace referencia.

Para aplicar la función PRUEBA.Z.N,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. haga clic en el icono Insertar función situado en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de funciones Estadísticas en la lista,
  4. haga clic en la función PRUEBA.Z.N,
  5. introduzca los argumentos requeridos separados con comas o seleccione un rango de celdas con el ratón,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
