Fonction PETITE.VALEUR

La fonction PETITE.VALEUR est l'une des fonctions statistiques. Elle est utilisée pour analyser une plage de cellules et renvoyer la k-ième plus petite valeur.

La syntaxe de la fonction PETITE.VALEUR est :

PETITE.VALEUR(array , k)

où

array est une plage de cellules à analyser.

k est la position du nombre à partir de la plus petite valeur, une valeur numérique supérieure à 0 saisie à la main ou incluse dans la cellule à laquelle il est fait référence.

Pour appliquer la fonction PETITE.VALEUR,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Statistique depuis la liste,
  4. cliquez sur la fonction PETITE.VALEUR,
  5. insérez les arguments requis en les séparant par des virgules,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
