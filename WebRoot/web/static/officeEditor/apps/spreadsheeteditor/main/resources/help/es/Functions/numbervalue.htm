Función VALOR.NUMERO

La función VALOR.NUMERO es una función de texto y datos. Se usa para convertir texto a un número, de forma local-independiente. Si el texto convertido no es un número, la función devolverá un error #VALUE!.

La sintaxis de la función VALOR.NUMERO es:

VALOR.NUMERO(texto [, [separador-decimal] [, [separador-grupal]])

donde

texto es datos de texto que representan un número.

separador-decimal es el carácter que se usa para separar la parte natural y fraccional del resultado. Es un argumento opcional. Si se omite, se usa el local actual.

separador-grupal es el carácter usado para separar grupos de números, como miles de cientos y millones de miles. Es un argumento opcional. Si se omite, se usa el local actual.

Los datos pueden ser introducidos manualmente o incluidos en las celdas a las que usted hace referencia.

Para aplicar la función VALOR.NUMERO,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione grupo de funciones Texto y datos en la lista,
  4. haga clic en la función VALOR.NUMERO,
  5. introduzca los argumentos correspondientes separados por comas,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
