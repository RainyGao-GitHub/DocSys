Funzione TIME

La funzione TIME è una delle funzioni della categoria Data e ora. Si usa per aggiungere un'ora particolare nel formato scelto (hh:mm tt di default).

La sintassi della funzione TIME è:

TIME(hour, minute, second)

dove

hour è un numero da 0 a 23.

minute è un numero da 0 a 59.

second è un numero da 0 a 59.

I valori numerici possono essere inseriti a mano o inclusi nella cella alla quale fate riferimento.

Per applicare la funzione TIME,

  1. selezionate la cella dove desiderate visualizzare il risultato,
  2. cliccate sull'icona Inserisci funzione sulla barra degli strumenti superiore,
    o cliccate con il tasto destro del mouse sulla cella scelta e selezionate l'opzione Inserisci funzione dal menu contestuale,
    o cliccate sull'icona prima della barra della formula,
  3. selezionate il gruppo di funzioni Data e ora dall'elenco,
  4. cliccate sulla funzione TIME,
  5. inserite gli argomenti richiesti separati da virgola,
  6. premete il tasto Enter.

Il risultato sarà visualizzato nella cella scelta.
