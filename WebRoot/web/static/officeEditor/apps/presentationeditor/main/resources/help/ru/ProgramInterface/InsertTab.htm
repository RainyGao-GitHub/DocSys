Вкладка Вставка

Вкладка Вставка позволяет добавлять в презентацию визуальные объекты и комментарии.

Окно онлайн-редактора презентаций:

Окно десктопного редактора презентаций:

С помощью этой вкладки вы можете выполнить следующие действия:

  • вставлять таблицы,
  • вставлять текстовые поля и объекты Text Art, изображения, фигуры, диаграммы,
  • вставлять комментарии и гиперссылки,
  • вставлять колонтитулы, дату и время, номера слайдов.
  • вставлять уравнения, символы.
