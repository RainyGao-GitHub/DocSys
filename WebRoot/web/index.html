系统简介

DocSys一个开源的文件管理系统，是用Java开发的WEB应用，采用的SSM框架，主要用于企业或个人的文件存储管理，方便随时查看和统一管理。

操作简单

本地化操作习惯：查看、编辑、拖拽、复制、粘贴、新建、删除。

使用方便

想看就看，想改就改，前提是网络要通还要有台电脑（也许手机也行）

功能并不强大

历史版本查看、用户管理、其他功能等着你去扩展。

代码开源

Java WEB项目、SSM框架、项目构建文档、详细设计文档。
